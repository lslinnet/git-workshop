push & pull

The pushing


git push [--all | --mirror | --tags] [--follow-tags] [-n | --dry-run]
         [--receive-pack=<git-receive-pack>] [--repo=<repository>]
         [-f | --force] [--prune] [-v | --verbose] [-u | --set-upstream]
         [<repository> [<refspec>...]]

Know where you push to!


git push -u [<repository> [<refspec>...]]
# or
git push origin master
especially with multiple remotes

The pulling


git pull [options] [<repository> [<refspec>...]]
